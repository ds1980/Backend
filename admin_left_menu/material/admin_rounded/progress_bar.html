Simple progress bar

To caption a progress bar, simply associate the value with the <progress> element.

<div class="simple-progress-bar">
    <progress class="progress" value="0" max="100"></progress>
    <progress class="progress" value="25" max="100"></progress>
    <progress class="progress" value="50" max="100"></progress>
</div>

Colored progress bar

To use alert class colors in a progress bars, use the class .progress-success, .progress-warning for colored styles.

 <div class="colored-progress-bar">
    <progress class="progress progress-success" value="25" max="100"></progress>
    <progress class="progress progress-info" value="50" max="100"></progress>
    <progress class="progress progress-warning" value="75" max="100"></progress>
    <progress class="progress progress-danger" value="90" max="100"></progress>
</div>

Striped progress bar

Add .progress-striped class to display striped progress bar. It uses a gradient to create a striped effect.

<div class="striped-progress-bar">
    <progress class="progress progress-striped progress-info" value="25" max="100"></progress>
    <progress class="progress progress-striped progress-warning" value="50" max="100"></progress>
    <progress class="progress progress-striped progress-danger" value="75" max="100"></progress>
    <progress class="progress progress-striped" value="90" max="100"></progress>
</div>

Animated progress bar

Show a progress bar animation with the use of progressbar plugin. 

<div class="animated-bar" id="line-progressbar" data-plugin="progressbar" data-type="Line" data-duration="3400" data-stroke-width="3" data-color="#087380" data-trail-color="#eee" data-animate="1.0" data-svg-style="{width: '100%', height: '100%'}"></div>
/* Use id alongwith different values to customize & create different animated progress bars */

Sizing progress bar

Use .progress-sm, .progress-md and .progress-lg classes to create progress bars in different sizes.

<div class="sizing-progress-bar">
    <progress class="progress progress-sm"  value="25" max="100"></progress>
    <progress class="progress progress-md  progress-warning" value="12" max="100"></progress>
    <progress class="progress progress-danger " value="60" max="100"></progress>
    <progress class="progress progress-lg progress-info" value="44" max="100"></progress>
</div>
