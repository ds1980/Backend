Basic alerts

Alerts are available for any length of text, as well as an optional dismiss button.

Primary Li Europan lingues es membres del sam.
Success Li Europan lingues es membres del sam.
Info Li Europan lingues es membres del sam.
Warning Li Europan lingues es membres del sam.
Danger Li Europan lingues es membres del sam. 
<div class="alert alert-primary" role="alert">
    <strong>Primary</strong> Li Europan lingues es membres del sam.
</div>

/* Use available contextul classes alert-success, alert-info, alert-warning, alert-danger to use alert with colors. */

Dismissing alerts

To add a dismiss button use .alert-dismissible class (optional) and for close button use .close.

Primary Li Europan lingues es membres del sam.
Success Li Europan lingues es membres del sam.
Info Li Europan lingues es membres del sam.
Warning Li Europan lingues es membres del sam.
Danger Li Europan lingues es membres del sam. 
<div class="alert alert-primary alert-dismissible fade in" role="alert">
    <button type="button" class="close" data-dismiss="alert" aria-label="Close">
        <span aria-hidden="true">×</span>
    </button>
    <strong>Primary</strong> Li Europan lingues es membres del sam.
</div>

/*  Use available contextul classes alert-success, alert-info, alert-warning, alert-danger to provide color effects in dismissable alerts. */

Link color

Use the .alert-link utility class to quickly provide matching links within any alert.

Well done! You successfully read this important alert message.
Heads up! This alert needs your attention, but it's not super important.
Warning! Better check yourself, you're not looking too good.
Oh snap! Change a few things up and try submitting again. 
<div class="alert alert-success" role="alert">
    <strong>Well done!</strong> You successfully read <a href="#" class="alert-link">this important alert message</a>.
</div>
